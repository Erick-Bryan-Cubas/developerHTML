Formatando textos: < strong >, < i >, < b > e < u >

O elemento < strong > é usado para destacar um texto.

O elemento < i > é usado para colocar um texto em itálico.

O elemento < b > é usado para colocar um texto em negrito.

O elemento < u > é usado para sublinhar um texto.

O elemento < s > é usado para riscar um texto.

O elemento < em > é usado para colocar um texto em ênfase.

O elemento < sup > é usado para colocar um texto em superior.

O elemento < sub > é usado para colocar um texto em inferior.

Tag < div > e < span >

A tag < div > é usada para criar uma divisão ou seção em um documento HTML.

A div é um tipo de tag display block, ou seja, ela ocupa toda a largura do seu container.

A tag < span > é usada para aplicar estilos a partes de um texto.
