Formatando textos: < strong >, < i >, < b > e < u >

O elemento < strong > é usado para destacar um texto.

O elemento < i > é usado para colocar um texto em itálico.

O elemento < b > é usado para colocar um texto em negrito.

O elemento < u > é usado para sublinhar um texto.

O elemento < s > é usado para riscar um texto.

O elemento < em > é usado para colocar um texto em ênfase.

O elemento < sup > é usado para colocar um texto em superior.

O elemento < sub > é usado para colocar um texto em inferior.

Tag < embed >

A tag é usada para incorporar conteúdo multimídia, como áudio ou vídeo, em um documento HTML.

Tag < audio >

A tag < audio > é usada para incorporar conteúdo de áudio, como música ou outros sons, em um documento HTML.

Flash Player

O Flash Player é um plugin que permite a execução de conteúdo multimídia, como animações e vídeos, em navegadores da web.

O plugin Flash Player foi descontinuado em 2020 e não é mais suportado pela Adobe.

Java Applet

A tag < applet > é usada para incorporar um applet Java em um documento HTML.

O applet a seguir exibe a mensagem "Hello, World!" na tela.

Era muito usado para criar jogos e animações, mas hoje em dia é pouco utilizado.

Seu navegador não suporta Java Applet.

Resenha de Cores

RGB (Red, Green, Blue) é um modelo de cor aditivo em que as cores vermelho, verde e azul são combinadas em várias intensidades para criar uma ampla gama de cores.

HEX (Hexadecimal) é um sistema de numeração posicional que representa números em base 16.

HSL (Hue, Saturation, Lightness) é um modelo de cor cromática que define as cores em termos de matiz, saturação e luminosidade.

CMYK (Cyan, Magenta, Yellow, Key) é um modelo de cor subtrativo usado para impressão em cores.

HSV (Hue, Saturation, Value) é um modelo de cor cromática que define as cores em termos de matiz, saturação e valor.

LAB (Lightness, a, b) é um modelo de cor cromática que define as cores em termos de luminosidade e duas coordenadas de cor.

XYZ é um modelo de cor cromática que define as cores em termos de tristimulus.

LCH (Lightness, Chroma, Hue) é um modelo de cor cromática que define as cores em termos de luminosidade, croma e matiz.

LUV (Lightness, u, v) é um modelo de cor cromática que define as cores em termos de luminosidade e duas coordenadas de cor.

YUV (Luminance, Chrominance) é um modelo de cor cromática que define as cores em termos de luminância e crominância.

YCbCr (Luminance, Chrominance Blue, Chrominance Red) é um modelo de cor cromática que define as cores em termos de luminância e duas coordenadas de cor.

YIQ (Luminance, In-phase, Quadrature) é um modelo de cor cromática que define as cores em termos de luminância e duas coordenadas de cor.

YPbPr (Luminance, Blue-difference, Red-difference) é um modelo de cor cromática que define as cores em termos de luminância e duas coordenadas de cor.

YDbDr (Luminance, Blue-difference, Red-difference) é um modelo de cor cromática que define as cores em termos de luminância e duas coordenadas de cor.

HSLA (Hue, Saturation, Lightness, Alpha) é um modelo de cor cromática que define as cores em termos de matiz, saturação, luminosidade e transparência.
