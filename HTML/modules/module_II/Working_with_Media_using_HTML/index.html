Trabalhando com Mídias utilizando HTML

O que são Tags de Mídia?

As tags de mídia são usadas para incorporar conteúdo multimídia, como áudio, vídeo e animações, em um documento HTML. As tags de mídia incluem < audio >, < video >, < embed >, < object >, < applet > e < iframe >. E são utilizadas para criar uma experiência de usuário mais rica e envolvente e para criar conteúdo interativo e dinâmico.

Tag < img >

A tag < img > é usada para incorporar imagens em um documento HTML. Dos atributos:
- src - especifica o URL da imagem.
- alt - especifica um texto alternativo para a imagem, se a imagem não puder ser exibida.
- width - especifica a largura da imagem, em pixels.
- height - especifica a altura da imagem, em pixels.
- style - especifica um estilo CSS para a imagem.
- usemap - especifica uma imagem como um mapa de imagem.
- ismap - especifica que a imagem é um mapa de imagem.
- loading - especifica como a imagem deve ser carregada quando a página é carregada.
- referrerpolicy - especifica a política de referência para a imagem.
- crossorigin - especifica uma política de CORS para a imagem.
- decoding - especifica como o navegador deve lidar com a decodificação da imagem.
- importance - especifica a importância da imagem.
- intrinsicsize - especifica as dimensões intrínsecas da imagem.
- loading - especifica como a imagem deve ser carregada quando a página é carregada.
- referrerpolicy - especifica a política de referência para a imagem.
- sizes - especifica os tamanhos de imagem responsivos.
- srcset - especifica as imagens a serem exibidas em diferentes situações.
- svg - especifica a imagem como um gráfico vetorial escalável.
- title - especifica um título para a imagem.

Tag < audio >

A tag < audio > é usada para incorporar arquivos de som em um documento HTML. Dos atributos:
- src - especifica o URL do arquivo de áudio.
- autoplay - especifica que o áudio deve começar a tocar assim que for carregado.
- controls - especifica que os controles de áudio devem ser exibidos (como reproduzir / pausar, volume, etc.).
- loop - especifica que o áudio deve começar de novo quando terminar.
- muted - especifica que o áudio deve ser reproduzido sem som.
- preload - especifica se e como o áudio deve ser carregado quando a página é carregada.
- volume - especifica o volume do áudio.
- crossorigin - especifica uma política de CORS para o arquivo de áudio.
- mediagroup - especifica um grupo de mídia ao qual o arquivo de áudio pertence.
- controlslist - especifica os controles que devem ser exibidos com o áudio.
- disableremoteplayback - especifica se o áudio deve ser reproduzido remotamente.
- intrinsicsize - especifica as dimensões intrínsecas do áudio.
- poster - especifica uma imagem para ser exibida enquanto o arquivo de áudio é baixado ou até que o usuário inicie a reprodução.
- preload - especifica se e como o áudio deve ser carregado quando a página é carregada.
- srcset - especifica os arquivos de áudio a serem exibidos em diferentes situações.
- type - especifica o tipo de arquivo de áudio.
- crossorigin - especifica uma política de CORS para o arquivo de áudio.
- referrerpolicy - especifica a política de referência para o arquivo de áudio.



Tag < video >

A tag < video > é usada para incorporar vídeos em um documento HTML. Dos atributos:
- src - especifica o URL do arquivo de vídeo.
- autoplay - especifica que o vídeo deve começar a tocar assim que for carregado.
- controls - especifica que os controles de vídeo devem ser exibidos (como reproduzir / pausar, volume, etc.).
- height - especifica a altura do vídeo em pixels.
- loop - especifica que o vídeo deve começar de novo quando terminar.
- muted - especifica que o vídeo deve ser reproduzido sem som.
- poster - especifica uma imagem para ser exibida enquanto o vídeo é baixado ou até que o usuário inicie a reprodução.
- preload - especifica se e como o vídeo deve ser carregado quando a página é carregada.
- width - especifica a largura do vídeo em pixels.
- crossorigin - especifica uma política de CORS para o arquivo de vídeo.
- mediagroup - especifica um grupo de mídia ao qual o arquivo de vídeo pertence.
- playsinline - especifica que o vídeo deve ser reproduzido no próprio quadro, em vez de em tela cheia.
- controlslist - especifica os controles que devem ser exibidos com o vídeo.
- disableremoteplayback - especifica se o vídeo deve ser reproduzido remotamente.
- intrinsicsize - especifica as dimensões intrínsecas do vídeo.
- srcset - especifica os arquivos de vídeo a serem exibidos em diferentes situações.
- type - especifica o tipo de arquivo de vídeo.

Tag < track >

A tag < track > é usada para especificar texto de legenda para um elemento < video >. Dos atributos:
- default - especifica que a legenda deve ser exibida por padrão.
- kind - especifica o tipo de legenda.
- label - especifica o idioma da legenda.
- src - especifica o URL do arquivo de legenda.
- srclang - especifica o idioma da legenda.

Tag < iframe >

A tag < iframe > é usada para incorporar outro documento HTML em um documento HTML. Dos atributos:
- src - especifica o URL do documento a ser incorporado.
- width - especifica a largura do iframe em pixels.
- height - especifica a altura do iframe em pixels.
- name - especifica o nome do iframe.
- sandbox - especifica se um iframe deve ser carregado em um sandbox.
- seamless - especifica que o iframe deve ser exibido sem bordas.
- srcdoc - especifica o conteúdo do documento a ser incorporado.
- allowfullscreen - especifica se o iframe pode ser exibido em tela cheia.
- allowpaymentrequest - especifica se o iframe pode fazer solicitações de pagamento.
- csp - especifica a política de segurança do conteúdo do documento a ser incorporado.
- referrerpolicy - especifica a política de referência para o documento a ser incorporado.
- loading - especifica como o iframe deve ser carregado quando a página é carregada.
- importance - especifica a importância do iframe.
- referrerpolicy - especifica a política de referência para o iframe.
