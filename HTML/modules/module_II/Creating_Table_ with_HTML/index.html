Criando Tabelas com HTML

O que é uma Tabela?

Uma tabela é uma coleção de dados organizados em linhas e colunas.
As tabelas são úteis para organizar e exibir dados, mas elas podem ser difíceis de ler em uma página da web.
Além de exibir dados em um formato mais organizado e fácil de ler. também é possível adicionar links, imagens e outros elementos a uma tabela.
