O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para documentos destinados a serem exibidos em um navegador web.

Tags Básicas Explicadas

<h1> até <h6> - Cabeçalhos

Estas tags definem cabeçalhos de diferentes níveis, com <h1> sendo o mais importante, e <h6>, o menos. São fundamentais para a estrutura e acessibilidade do conteúdo.

<p> - Parágrafos

Usada para definir parágrafos de texto. É um dos elementos mais comuns, estruturando o texto em blocos separados para melhor leitura.

<mark> - Marcação

Destaca partes do texto, geralmente com um fundo amarelo, para indicar relevância ou para chamar a atenção para essa seção.

<small> - Texto Pequeno

Diminui o tamanho do texto, útil para disclaimers, notas de rodapé, ou comentários secundários.

<i> - Itálico

Representa texto em itálico, geralmente usado para ênfase ou para títulos de obras.

<u> - Sublinhado

Adiciona um sublinhado ao texto. Embora tradicionalmente usado para ênfase, o uso de CSS para estilização é preferível para evitar confusão com links.

<strong> - Negrito

Indica que o texto é de importância significativa, aumentando o peso (bold) para destaque.

<ol> - Lista Ordenada

Cria uma lista de itens numerados, útil para classificações, instruções passo a passo, ou quando a ordem dos itens é importante.

<ul> - Lista Não Ordenada

Cria uma lista de itens sem uma ordem específica, ideal para listagens onde a sequência não é relevante.

<li> - Item de Lista

Define um item dentro de uma lista ordenada (<ol>) ou não ordenada (<ul>). É um elemento filho de listas.

<a> - Âncora (Link)

Define um hyperlink para outro documento ou para uma seção dentro do mesmo documento. Essencial para a navegação na web.

<hr> - Linha Horizontal

Insere uma quebra visual na forma de uma linha horizontal. É útil para separar seções de conteúdo.

<sub> - Subscrito

Exibe o texto em um tamanho menor abaixo da linha de base do texto. Comumente usado em fórmulas químicas ou matemáticas.

<sup> - Sobrescrito

Exibe o texto em um tamanho menor acima da linha de base do texto. Frequentemente usado para notas de rodapé ou datas.

<blockquote> - Citação em Bloco

Usado para citar trechos de texto de outra fonte, indicando visualmente (geralmente com indentação) que o texto é uma citação longa.

<font> - Fonte (Obsoleta)

Antigamente usada para definir o tipo de letra, tamanho e cor do texto. Agora, esses estilos são controlados via CSS.

<del> - Deletado

Indica texto que foi excluído ou é irrelevante. O texto aparece riscado.

<abbr> - Abreviação

Define uma abreviação ou acrônimo; a explicação completa pode ser dada no atributo title.
